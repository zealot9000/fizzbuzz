# Fizzbuzz-gem

It becomes easier - to play FizzBuzz right now.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'fizzbuzz_zealot9000' 
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install 'fizzbuzz_zealot9000'

## Usage

    $ require 'fizzbuzz'
    $ puts fizzbuzz 5

## Test

    $ rake spec_all
